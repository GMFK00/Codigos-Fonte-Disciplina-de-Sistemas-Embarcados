Módulo RTC DS3231 Protocolo: I2C • 3.3V - 5V

Pinagem do Módulo (ZS-042)
32K 32kHz Out Saída de clock de precisão (Geralmente não usado). CLOCK
SQW Interrupt Square Wave / Output Alarm. Usado para acordar o ESP32 (Wakeup). ALARM / INT
SCL Serial Clock Clock do barramento I2C. I2C SCL
SDA Serial Data Dados do barramento I2C. I2C SDA
VCC Power Alimentação (3.3V recomendado para ESP32). POWER
GND Ground Referência de terra. GND
Nota Importante: O pino SQW é Active-LOW (vai para GND quando o alarme dispara). Use INPUT_PULLUP no microcontrolador ou um resistor de pull-up externo.

Módulo RF nRF24L01+ PA/LNA Protocolo: SPI • 3.3V APENAS

Pinagem (Header 2x4)
1. GND Ground Referência de terra (Borda Branca no módulo). GND
2. VCC 3.3V Input Alimentação. ⚠ NÃO LIGAR EM 5V. MAX 3.6V
3. CE Chip Enable Controla modo RX/TX. (GPIO Digital). CONTROL
4. CSN Chip Select Seleção do escravo SPI (Active LOW). SPI SS
5. SCK SPI Clock Clock do barramento SPI. SPI CLK
6. MOSI Master Out Dados saindo do ESP32 para o Rádio. SPI MOSI
7. MISO Master In Dados saindo do Rádio para o ESP32. SPI MISO
8. IRQ Interrupt Opcional. Sinaliza recebimento de dados. INT
Dica de Hardware: É altamente recomendado soldar um capacitor eletrolítico (10uF a 100uF) diretamente entre os pinos VCC e GND do módulo para evitar falhas de transmissão.

Atenção: A pinagem do DHT11 pode variar dependendo do fabricante do módulo (PCB). Sempre verifique as etiquetas (+, -, S) na placa.
