Glossário de Tags e Funcionalidades

ADC Analog-to-Digital Converter

Conversores que leem tensões analógicas (0V a 3.3V) e transformam em valores digitais (0 a 4095).
• ADC1: Pode ser usado livremente.
• ADC2: Restrito! Não pode ser usado se o Wi-Fi estiver ligado.

TOUCH Capacitive Touch

Pinos com circuito interno para detectar toque humano (capacitância), substituindo botões físicos.
Pode acordar o ESP32 do modo Deep Sleep. Sensibilidade ajustável via software.

RTC Real Time Clock GPIO

Pinos conectados ao subsistema de Baixo Consumo (Low Power).
Únicos pinos que funcionam (leitura/escrita) quando o chip está dormindo (Deep Sleep).

I2C Inter-Integrated Circuit

Protocolo de comunicação serial de dois fios (SDA/SCL) para sensores e displays.
• SDA: Serial Data
• SCL: Serial Clock
*Nota: O ESP32 permite mapear I2C em qualquer pino, mas os marcados na imagem (8/9) são os padrões sugeridos.

FSPI/SPI Serial Peripheral Interface

Protocolo de alta velocidade (Full Duplex) para cartões SD, Displays TFT e memórias Flash externas.
• FSPI: Fast SPI (Controlador dedicado de alta velocidade).
• VSPI/HSPI: Nomenclaturas legadas do ESP32 clássico, ainda usadas.
• Pinos: MOSI (Data Out), MISO (Data In), CLK (Clock), CS/SS (Select).

UART Universal Asynchronous RX/TX

Interface Serial padrão.
• UART0 (TXD0/RXD0): Dedicada ao upload de código e monitor serial (USB-Serial converter).
• UART1/2: Livres para conectar GPS, módulos GSM, etc.
• RTS/CTS: Pinos de controle de fluxo de hardware.

USB Native USB OTG

Conexão USB direta ao silício do ESP32-S3 (sem chip conversor CH340/CP2102).
Permite que o ESP32 simule teclados (HID), mouses, pendrives (MSC) ou porta serial virtual (CDC).

JTAG Hardware Debug

Interface padrão da indústria para debugar código linha a linha.
Permite usar breakpoints no VS Code/PlatformIO sem usar `Serial.print`. Pinos: MTDI, MTCK, MTMS, MTDO.

STRAP Strapping Pins

Pinos críticos que decidem como o chip liga (Boot Mode, Voltagem da Flash, etc.).
⚠ Cuidado: Evite puxar esses pinos para HIGH ou LOW externamente durante a inicialização, ou o ESP32 pode não ligar.
Pinos: 0, 3, 45, 46.

SYS System / Clock

Funções utilitárias de sistema.
• XTAL_32K: Cristal externo para relógio preciso.
• CLK_OUT: Saída de sinal de clock.
• RGB LED: Pino do LED endereçável integrado.
